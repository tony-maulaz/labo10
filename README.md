# labo10

## Tests

Lors de ce laboratoire, vous allez devoir tester le programme d'un de vos collègues.

Des binômes seront formés pendant le laboratoire et vous devrez contrôler le programme d'une
autre personne.

Vous devrez rendre un rapport de test sur le programme que vous avez reçu. *Idem que pour le labo 8.*

### Échange du programme

Pour vous échangez le programme, il vous suffit de copier le fichier `string` que vous avez compilé.

Copier le dans votre répertoire de laboratoire sous un autre nom *(ex: stringtest)*.

Ensuite vous pouvez l'executer en entrant `./stringtest count "foo bar"`.

Pour l'executer il faut le faire dans `VS Code` avec le `container` ouvert.
